# Installation

## From GitHub Releases

1. Download the [latest release](https://github.com/moongdevstudio/AssetPlus/releases)
2. Extract the ZIP file
3. Copy the `addons/assetplus` folder to your project's `addons` folder
4. Enable the plugin in **Project Settings > Plugins > AssetPlus**

## From Godot Asset Library

!!! note "Coming Soon"
    AssetPlus will be available on the Godot Asset Library soon.

## Verify Installation

After enabling the plugin, you should see **AssetPlus** in the top toolbar, next to 2D, 3D, and Script tabs.

## First Launch

On first launch, AssetPlus will show a welcome guide explaining the main features. You can access this guide again from the Settings.

## Updating

AssetPlus includes an auto-update feature that checks for new versions at startup. When an update is available:

1. A dialog will appear with release notes
2. Click **Install Now** to update automatically
3. The editor will restart with the new version

You can disable auto-update checks in [Settings](../reference/settings.md).
